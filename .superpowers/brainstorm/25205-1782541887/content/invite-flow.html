How do enterprise members join a team?

The manager pays and controls seats — but how do team members actually get added?

A

Email invite

Manager types email addresses → each person gets an invite link → they sign up (or log in) and are automatically joined to the team. Most common SaaS pattern.

B

Invite link / code

Manager generates a shareable link or code → shares it however they like (Slack, email, etc.) → anyone with the link joins the team up to the seat limit.

C

Domain-based auto-join

Manager sets a company email domain (e.g. acme.com) → any DealEcho user with that domain is automatically added, or gets a prompt to join.
