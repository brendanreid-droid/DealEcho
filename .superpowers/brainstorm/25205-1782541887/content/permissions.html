What can a manager control per team member?

Select all that apply — these are the per-seat permissions the manager can set

A

Role: Manager vs User

Promote a user to manager (can also manage seats/team) or demote back to user.

B

Enable / disable seat

Deactivate a member without removing them — they lose Pro access but keep their account. Seat stays allocated.

C

Remove from team

Free up their seat entirely. They revert to free tier. Seat can be reassigned.

D

Custom feature flags

Per-user toggles for specific features (e.g. can submit reviews, can export data). More granular than role.
